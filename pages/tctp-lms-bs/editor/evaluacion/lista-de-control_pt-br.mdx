# Lista de control

## Conocer lo básico

Una lista de control (checklist), es una lista de acciones que requieren completarse y se utiliza para resaltar aspectos importantes del curso. 
Los usuarios pueden usar esto como una guía visual para determinar sus necesidades para completar el curso.

Este tutorial muestra cómo crear una lista de control.
