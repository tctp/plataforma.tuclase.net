# LMS Brightspace
La plataforma que utiliza Tu clase, tu país es un ambiente de aprendizaje denominado por sus siglas como LMS (Learning Management System), este sistema fue desarrollado por la empresa Brightspace D2L de origen Canadiense, partners tecnológicos de Tu clase, tu país.
El entorno de aprendizaje es un innovador sistema de gestión educativa para crear, alojar y editar recursos de aprendizaje en línea. Contiene una serie de herramientas flexibles para crear contenido personalizado para instituciones, cursos y usuarios.

Actualmente tenemos diferentes instancias operativas con este sistema:

- <a href="https://cursos.tuclase.net" target="_blank">TCTP</a>
- <a href="https://modulos.programaeleva.cl" target="_blank">Eleva</a>
- <a href="https://cursos.modelopionero.cl" target="_blank">Modelo pionero</a>
- <a href="https://inetcampus.educacion.gob.ar" target="_blank">INET</a>
- <a href="https://espaciomaestro.educacionbogota.edu.co" target="_blank">Espacio Maestro</a>
