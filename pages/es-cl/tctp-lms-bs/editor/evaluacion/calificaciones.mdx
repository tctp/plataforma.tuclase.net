# Calificaciones

## Conocer lo básico

La herramienta calificaciones permite configurar y administrar, esquemas y libros de calificaciones, para evaluar diferentes actividades de la plataforma. 

Con esta herramienta puedes:

1. Calcular notas
2. Asociar elementos de calificación
3. Crear esquemas de calificación
4. Vincular recursos o actividades
5. Crear categorias de calificación
6. Entre otros.

Este tutorial describe los aspectos clave de la creación y el uso de su libro de calificaciones.

## Crear un elemento de calificación
Los elementos de calificación representan el trabajo con el que se evalúa a los usuarios en un curso. 
Estos pueden existir de manera discrecional (sin asociación directa a un recurso del curso) o los puede asociar a elementos de calificación numérica con recursos del curso como: debates, cuestionarios y buzón. 
Cada elemento de calificación tiene una entrada en el libro de calificaciones, a la que le asigna una calificación para cada usuario. Dependiendo del tipo de elemento de calificación que quiera crear, los elementos de calificación se pueden calificar de forma numérica o según un esquema de calificaciones. 
Este tutorial muestra cómo crear un nuevo elemento de calificación en el libro de calificaciones.

## Crear una categoria para elementos de calificación
Las categorías de calificación organizan y agrupan en secciones, los elementos de calificaciones relacionadas en su libro de calificaciones. Por ejemplo, una categoría de calificación de "Tareas de redacción" para agrupar elementos del buzón y una categoría de calificación de "Participación en clase" para agrupar los elementos de debate. Las categorías de calificación aparecen en **"Administrar calificaciones"**. Este tutorial te muestra cómo crear una categoría en su libro de calificaciones.
