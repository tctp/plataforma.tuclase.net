# Catálogo de cursos

El Catálogo de cursos es un sistema que se integra a la plataforma LMS mediante la autenticación por SSO (single sign on o Inicio de Sesión Único. Es un procedimiento de autenticación que habilita a un usuario determinado para acceder a varios sistemas con una sola instancia de identificación).
Este sistema muestra una lista de cursos y programas activos para la organización, los cuales, se pueden configurar a través de su panel de administración.
Además este sistema permite configurar la inscripción a los usuarios en cursos y programas.
